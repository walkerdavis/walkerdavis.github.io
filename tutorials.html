Creative Side-Chaining and Envelope Following Part 4: Filtering

The Auto Filter does not provide any visual feedback of its own, so I will have Ableton’s Spectrum Audio Effect/Tool to illustrate what I am demonstrating.
(The Auto Filter in Ableton 9.7 does have a ‘Drive’ parameter that does effect Gain/Amplitude when you select something other than the ‘Clean’ Filter Type, but I will not be discussing it here.)


1. Configure ‘Chainer’ and ‘Chained’ signal(s).
-Sounds that have short attack-times, like kick and snare drums, work really well for ‘Chainer’ signals.
-Longer sounds like guitar, keyboards, synths, and reverbs work well as ‘Chained’ signals.
-Grouping multiple Audio and MIDI tracks together, Command-G, will allow you to make the effect more obvious.

2. Put the Auto Filter Audio Effect on the ‘Chained’ track or group

3. Routing within the Auto Filter
-Press the Arrow icon in the top left of the Audio Filter Audio Effect
-Press the ‘SideChain’ button, it is On when the button is Yellow
-In the dropdown menu, select the ‘Chainer’ track

4. Input Parameters
Gain will control how much the ‘Chainer’ signal effects the Auto Filter

5. Filter Parameters
Frequency specifies the Cutoff Frequency of the Auto Filter before the Side-Chain processing happens(or if Side-Chain is turned off)
Resonance adjusts the Gain Boost at the Cutoff Frequency
-Both of these parameters can be adjusted with the knobs, or by dragging the Yellow ball around the grid

6. Envelope Parameters
Envelope specifies how far the Cutoff Frequency will be moved
-When at 0, top and center, the Auto Filter will not react or move
-When moved counterclockwise towards -127, the Cutoff Frequency will be decreased
-When moved clockwise towards 127, the Cutoff Frequency will be increased
Attack specifies how long it will take the Cutoff Frequency to move the amount specified by the ‘Envelope’ parameter
Release specifies how long it will take the Cutoff Frequency to return to the position specified by ‘Frequency’ and ‘Resonance’ parameters

7. Experiment!
Throw a low-pass filter on the master track and side-chain it to the kick!

Creative Side-Chaining and Envelope Following Part 3: Gating

1. Configure ‘Chainer’ and ‘Chained’ signal(s).
Sounds that have short attack-times, like kick and snare drums, work really well for ‘Chainer’ signals.
Longer sounds like guitar, keyboards, synths, and reverbs work well as ‘Chained’ signals.
Grouping multiple Audio and MIDI tracks together, Command-G, will allow you to make the effect more obvious.

2. Put the Gate Audio Effect on the ‘Chained’ track or group

3. Routing within the Gate
Press the Arrow icon in the top left of the Gate Audio Effect
Press the ‘SideChain’ button, it is On when the button is Yellow
In the dropdown menu, select the ‘Chainer’ track

4. Input Parameters
Threshold determines the minimum amplitude value at which the Gate will open
Return sets the value at which the Gate will close. This amplitude value is relative to the Threshold value,
Flip reverses the Gate’s On/Off status. When the input/‘Chainer signal exceeds the set Threshold, the Gate is now turned Off instead of On

5. Envelope Parameters
Attack specifies how long it will take for the ‘Chained’ signal to reach its full amplitude
Hold specifies how long this amplitude will be sustained
Release specifies the time it will take for the amplitude to return to its ‘Floor’ value once the Gate has been closed
Floor specifies a resting amplitude for the ‘Chained’ signal. If you do not want to hear it at all, set it to negative infinity.

6. Experiment!
Set up multiple ‘Chained’ tracks/groups, put identical Gates on each of them(Copy and Paste with Command-C), select Flip on some of them...


Traditionally, gates are used to clean up an electric signal by removing the low-level hum that is present whenever a noisy guitar or keyboard is plugged in. This is done by setting the amplitude threshold of the gate above the amplitude of this hum. When the player strums, plucks, or hits a key, the instrument’s output volume, a significantly louder sound than that of the hum, open’s the gate and the signal is let through. Similar to the Side-Chain Compression tutorial, the method I described is more of a creative technique that might get you some interesting results.

Creative Side-Chaining and Envelope Following Part 2: Compression

1. Configure ‘Chainer’ and ‘Chained’ signal(s).
-Sounds that have short attack-times, like kick and snare drums, work really well for ‘Chainer’ signals.
-Longer sounds like guitar, keyboards, synths, and reverbs work well as ‘Chained’ signals.
-Grouping multiple Audio and MIDI tracks together, Command-G, will allow you to make the effect more obvious.

2. Put a Compressor Audio Effect on the ‘Chained’ track or group

3. Routing within the Compressor
Press the Arrow icon in the top left of the Compressor Audio Effect
Press the ‘SideChain’ button, it is On when the button is Yellow
In the dropdown menu, select the ‘Chainer’ track

4. Compression Parameters
Threshold sets the minimum amplitude value that will be compressed. Frequency content from the ‘Chained’ signal that is louder than the ‘Threshold’ value will have compression applied to it.
Output applies gain to the signal after it has been compressed
Ratio determines how much compression will be applied when the signal’s amplitude is above the threshold

5. Envelope Parameters
Attack specifies how fast the amplitude/output level of ‘Chained’ signal will be decreased
Release specifies how long it will take the ‘Chained’ signal to return to normal amplitude/output

6. Input Parameters
Gain adjusts how much the ‘Chainer’ signal will affect the Compressor
EQ allows you to specify which frequencies of the ‘Chainer’ signal will apply Side-Chain compression
-To Side-Chain everything to a kick drum, Select the Low-Pass Filter Type and set the Cutoff Frequency to 100 Hz

7. Experiment!
For very extreme effects, you can side-chain the master stereo output track to a single track!
You can also mute the ‘Chainer’ track


Traditionally, side-chain compression has been used to achieve greater control in a mix. For instance, side-chaining the bass to the kick drum prevents the mix from becoming muddy and potentially out of phase in that frequency area. The approach I described above is more of a creative and reactive approach that might give you unnatural and extreme results. I would approach side-chain compression for mixing purposes a little bit differently. (Hopefully that tutorial will be ready soon!)

Creative Side-Chaining and Envelope Following Part 1: Overview

These videos will show you the different types of side-chaining and envelope-following you can do in Ableton Live 9.7.

This video was written for Ableton 9.7; previous (and future) versions may require a few different steps.


Part 1: Three side-chaining methods

Side-chaining (or envelope-following) involves manipulating one audio source, with the amplitude of another audio source. The term ‘side-chaining’ is used when dealing with compressors, gates, or filters. Side-chaining has traditionally been used to add more control to a mix, but it can also be used to make very interesting, reactive, and unnatural things happen.

For the rest of these videos, I will refer to the audio signal being affected by side-chaining as the ‘Chained’ signal, and the influencing audio signal as the ‘Chainer.’ The Chained signal is affected/influenced/manipulated by the Chainer.


Side-chain compression: Compressors limit the dynamic range of an audio signal by decreasing the louder amplitudes and increasing the quieter amplitudes. When side-chained compression is applied, the Chained signal’s amplitude is temporarily decreased as the Chainer’s amplitude is louder than a specified threshold.

Gating: Gates operate like on/off switches that use a signal’s amplitude to determine the switch’s on/off status. The user specifies an amplitude threshold, which determines if the audio signal will be passed through or not. When the audio’s amplitude is below the threshold it will not be passed through, but if its amplitude is above the threshold, it will. When a Chained audio signal is ‘Gated’ to a different Chainer signal, the chained signal will be switch on or off based on the Chainer’s amplitude.

Filtering: Filters (called the Auto Filter in Ableton) decrease the amplitude of an audio signal’s specified frequency ranges. High-pass filters decrease the amplitude of frequencies below a specified cutoff, allowing only the frequencies above the cutoff to be sounded. Low-pass filters do the opposite, decreasing the amplitude of frequencies above the cutoff. The cutoff of a filter on a Chained audio signal can be increased or decreased temporarily based on the amplitude of Chainer's signal.

Quantizing MIDI and Adjusting the Quantization Settings

This video will show you how to quantize MIDI notes and how can adjust your quantization settings.

This video was written for Ableton 9.7; previous (and future) versions may require a few different steps.


1. Load a MIDI instrument and MIDI clip you would like to edit.

2. Select the portion you would like quantized
-Click and drag will select only the highlighted notes
-Command-A will select all notes within the clip

3. Quantize by pressing Command-U

4. Open you Quantization Settings by pressing Shift-Command-U

5. Configure these parameters:

Quantize To: Gives you a dropdown menu of the available grids
-I would leave this setting as ‘Current Grid’ and use the Command 1, 2 and 3 Ableton Shortcuts
-Command-1 Narrows the MIDI Grid
-Command-2 Widens the MIDI Grid
-Command-3 Toggles the Grid to Triplet Divisions

Adjust Note: Allow you select which part(s) of the part of the MIDI you would like to quantize.
Generally, you will want to quantize the ‘Start’ of the MIDI notes only

Amount: This specifies the percentage at which the selected MIDI notes will move towards the grid you are quantizing to.
0% would not shift the MIDI notes at all
100% snaps all MIDI notes exactly to the selected grid

I like to keep mine at 50%. This allows me to find a nice balance between my raw(perhaps sloppy), unquantized playing, and the rigid, robotic effect of complete quantization. Keeping the ‘Quantization Amount’ at 50% also allows you to easily get closer and closer to your desired feel by pressing Command-U a few times, instead of opening your Quantization Settings every time.

Recording MIDI Notes after the Arpeggiator

This video will show you how to record the MIDI notes after they have been arpeggiated. By default, Ableton records the input note and parameter information, not the notes that are being generated by the Arpeggiator MIDI Effect. This technique can be applied to Ableton’s other MIDI Effects too.

This video was written for Ableton 9.7; previous (and future) versions may require a few different steps.


1. Load a MIDI track, a MIDI instrument, and an Arpeggiator
-I will be using Ableton’s Operator Synth for this tutorial

2. Find a Arpeggiator Setting you like, record a MIDI clip
-Notice that the MIDI clip contains only the held notes, not the faster notes we are hearing

3. Add another MIDI track
-Rename tracks to ‘Sender’ and ‘Receiver’

4. Separate the MIDI Effect from the Instrument and Audio Effects
-Remove the MIDI Instrument and any Audio Effects from the ‘Sender’ track
-Copy them to the ‘Receiver’ MIDI track
-Make sure the Arpeggiator is on the ‘Sender’ track

5. Internal MIDI Routing
-In the ‘MIDI To’ dropdown menu, select ‘Receiver’
-Select ‘In’ for the Monitoring Mode
-This sets the ‘Receiver’ track to constantly listen for any MIDI information coming from the ‘Sender’ track

6. Arm and Record
-You can record the MIDI clips before and after the Arpeggiator

Saving Custom Keyboard Shortcuts

This video will show you how to create custom keyboard shortcuts and then save them in a Default Set for future use. Customizing your keyboard shortcuts in Ableton Live will allow to speed up your work flow. Here are the shortcuts I have used and found incredibly helpful over the past few years:

“1 - 9” - Trigger correlating scene
“ , ” comma - Stop playback
“ . ” period - Stop all Clips in Session View
“ / ” forward slash - Back to Arrangement View
“ ‘ ” apostrophe - Reenable Automation in Arrangement View
“ ; ” semi-colon - Loop Region On/Off
“ t ” - Metronome On/Of
“ y ” - Tap Tempo

This video was written for Ableton 9.7; previous (and future) versions may require a few different steps.


1. Open up a Blank Session.
-Do NOT ignore this step!

2. Create the Audio, MIDI, and Return tracks you would like in Default Session

3. Open up ‘Key May Mode’
-Press the 'Key' icon or press ‘Command-K’ to view parameters you can give shortcuts

4. Assign keyboard shortcuts
-Select item you want a shortcut for
-Press keyboard key you want to use as a shortcut
-Repeat until you have all keyboard shortcuts you like

5. Save this set as you default
-Open Ableton Preferences via the MenuBar or by pressing ‘Command- ,’
-Select the ‘File Folder’ tab on the left
-Click ‘Save’ for the ‘Save Current Set as Default Set’ option


Things to consider:

If you use Ableton’s ‘Computer MIDI Keyboard’ Mode (Shift-Command-K) to input MIDI notes, consider how a mis-typed key might affect your workflow. For instance, if you make ‘ q ’ the keyboard shortcut to ‘Stop’ global playback, accidentally pressing ‘ q ’ when using ‘Computer MIDI Keyboard’ Mode to play/input notes ‘ a ’ or ‘ w ’ will stop playback and potentially ruin a good take if you are recording.

When you save the current set to you ‘Default Set,’ only new projects will contain these custom keyboard shortcuts. If you would like to use them on existing projects, you will need to add them manually again. Ableton does this to protect you from deleting shortcuts or MIDI mappings you may have made in that particular project.

Splitting a Stereo Track to Left and Right Channels

This Ableton Project and Audio Effect Rack will allow you to split a stereo file, isolating the right and left channels. This video was written for Ableton 9.7; previous (and future) versions may require a few different steps.

1. Import your stereo audio file to an empty Audio Track

2. Setup the Tracks
-Create 3 Audio Tracks; rename them 1-Stereo, 2-Left and 3-Right

3. Create reusable ‘Stereo Splitter’ Audio Effect Rack
-Bring in an Empty Audio Effect Rack to the 1-Stereo Track, rename it ‘Stereo Splitter’
-Create Two “Chains” by dragging in two instances of Ableton’s Utility Audio Effect, rename them ‘Left’ and ‘Right’
-In First Chain/Utility, select ‘Left’ in the Utility-Channel Mode dropdown menu
-In Second Chain/Utility, select ‘Right’ in the Utility-Channel Mode dropdown menu
-(optional) Press the Floppy Disc Icon and save to your User Library

4. Internal Routing
-On the 2-Left track, adjust both ‘Audio From’ dropdown menus
-From ‘Ext. In’ to ‘1-Stereo’
-From ‘1’ to ‘Stereo Splitter | Left | Post FX’
-Set the Channel Monitor to ‘Auto’

-On the 3-Right track, adjust both ‘Audio From’ dropdown menus
-From ‘Ext. In’ to ‘1-Stereo’
-From ‘1’ to ‘Stereo Splitter | Right | Post FX’
-Set the Channel Monitor to ‘Auto’

5. Pan and Arm 2-Left and 3-Right tracks
-Hard pan the 2-Left completely to the left, and the 3-Right track completely to the right
-To arm multiple tracks, Command-click

6. Check to make sure everything is working
-Mute the 1-Stereo track
-If 2-Left and 3-Right do not play, or if they are the same, you have likely made an error somewhere in the steps above

7. Record
-In 2-Left and 3-Right, set empty clips to record
-Launch you stereo file/clip on 1-Stereo


Things to consider:

This is not a perfect solution. This method does not leave you with two mono tracks, but rather gives you two distinct stereo files that contain identical right and left information. Notice that both the 2-Left and 3-Right tracks display stereo meters, not a single mono meter.

Consider using Audacity, it allows you to split stereo files to mono files with a single right-click.
